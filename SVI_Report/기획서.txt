=========================================
사회적가치지표(SVI) 측정 설문 및 보고서 시스템
기획서 - 무채색 모던 디자인
=========================================

[프로젝트 개요]

• 프로젝트명: 사회적가치지표 포털 - 자가진단 및 보고서 시스템
• 디자인 컨셉: 미니멀 무채색 모던 디자인
• 목적: 사회적기업의 사회적가치 측정을 위한 온라인 설문조사 및 결과 보고서 시스템 구축
• 주요 기능:
  - 사용자/관리자 구분 로그인
  - 14개 지표 기반 설문 입력
  - 관리자 설문 리스트 관리
  - 측정 결과 보고서 자동 생성 및 조회


=========================================
[디자인 시스템 - 무채색 계열]
=========================================

■ 색상 팔레트 (Grayscale Only)
  • Black Tones
    - Pure Black: #000000 (Primary, Accent, CTA)
    - Dark Gray: #1a1a1a (Headers, Important Text)
    - Medium Dark: #2a2a2a (Hover States)
    - Gray: #333333 (Body Text)

  • Gray Scale
    - #4a4a4a (Secondary Text)
    - #666666 (Tertiary Text)
    - #888888 (Disabled Text)
    - #aaaaaa (Placeholders)

  • Light Tones
    - #cccccc (Borders Light)
    - #dddddd (Borders)
    - #e5e5e5 (Dividers)
    - #eeeeee (Backgrounds)
    - #f5f5f5 (Page Background)
    - #fafafa (Input Background)
    - #ffffff (White)

  • Status Colors (Monochrome)
    - Success/Complete: #000 (Black)
    - Warning/Pending: #888 (Gray)
    - Error: #333 (Dark Gray)
    - Info: #555 (Medium Gray)

■ 타이포그래피
  • 폰트: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
  • Weight: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
  • 크기
    - Large Headings: 32px, 28px, 24px
    - Headings: 20px, 18px, 16px
    - Body: 15px, 14px
    - Small: 13px, 12px
  • 줄간격: 1.6 (본문), 1.2 (제목)
  • Letter Spacing: -0.5px ~ -0.3px (제목), 0.5px ~ 1px (라벨)

■ 레이아웃
  • Border Radius: 0 (모든 요소 - 완전한 직각)
  • 간격
    - Section: 40px, 48px, 60px
    - Element: 16px, 20px, 24px, 32px
    - Component: 8px, 12px, 16px
  • 패딩
    - Large: 48px, 60px
    - Medium: 24px, 32px, 40px
    - Small: 12px, 16px, 20px

■ 그림자 (Subtle)
  • Soft Shadow: 0 0 60px rgba(0, 0, 0, 0.08)
  • Medium Shadow: 0 2px 8px rgba(0, 0, 0, 0.1)
  • Strong Shadow: 0 0 60px rgba(0, 0, 0, 0.3)
  • Overlay: rgba(0, 0, 0, 0.75)

■ 모션
  • Transition: all 0.2s ease
  • Hover Transform: scale(0.98)
  • Animation: spin (0.8s linear infinite)


=========================================
[1. 로그인 페이지]
=========================================

[화면 개요]
사용자 유형(관리자/사용자)을 선택하여 로그인하는 미니멀 페이지

[디자인 특징]
- 완전 무채색 (흑백 + 그레이스케일만 사용)
- Border Radius: 0 (직각 박스)
- 미니멀 레이아웃
- 넓은 여백과 간격
- 선명한 흑백 대비

[주요 구성 요소]

1. 로고 섹션
   - 검정 정사각형 로고 (64x64px)
   - 타이틀: 검정색, 700 weight
   - 서브타이틀: 회색, 대문자, letter-spacing

2. 사용자 유형 선택
   - 2개 그리드 레이아웃
   - 기본: 흰 배경, 회색 테두리
   - 선택: 검정 배경, 흰 텍스트
   - Border Radius: 0

3. 로그인 폼
   - 입력 필드: 회색 배경(#fafafa)
   - 포커스: 검정 테두리
   - Border Radius: 0
   - 버튼: 검정 배경, 흰 텍스트

4. 에러 메시지
   - 회색 배경
   - 좌측 검정 굵은 선

[기능 명세]
(이전과 동일)

[2. 설문 입력 페이지]
=========================================

[디자인 특징]
- 헤더: 흰 배경, 얇은 회색 하단 선
- 사이드바: 흰 배경, 회색 구분선
- 활성 항목: 검정 배경, 흰 텍스트
- 프로그레스 바: 검정 그라데이션
- 버튼: 검정 (다음/제출), 흰+검정테두리 (저장/이전)

[색상 매핑]
- Primary Action: #000 (검정)
- Secondary Action: White + Black Border
- Disabled: #ccc (회색)
- Hover: #2a2a2a (다크그레이)
- Active/Selected: #000 (검정)
- Background: #f5f5f5 (라이트그레이)

[섹션 헤더]
- 배경: #1a1a1a (다크 그레이)
- 텍스트: 흰색
- Border Radius: 0

[입력 요소]
- Input: #fafafa 배경, #e5e5e5 테두리
- Focus: #000 테두리
- Select/Dropdown: 동일
- Radio/Checkbox: 검정 선택 시

[테이블]
- 헤더: #f8f9fa 배경
- 테두리: #e0e0e0
- Hover: #f8f9fa

[버튼]
- 다음: #000 배경, 흰 텍스트
- 저장: 흰 배경, #000 텍스트, #000 테두리
- 이전: 흰 배경, 회색 텍스트, 회색 테두리
- 행추가: #000 배경

[액션 버튼]

[페이지네이션]
- 기본: 흰 배경, 회색 테두리

  - 등급2: #666 (미디엄그레이)

  - 등급4: #aaa (라이트그레이)

=========================================
[공통 디자인 원칙]
=========================================

1. **완전한 무채색**
   - 컬러 사용 금지
   - 오직 흑백과 그레이스케일만
   - 차트도 그레이스케일

2. **직각 디자인**
   - Border Radius: 0 (모든 요소)
   - 완전한 사각형
   - 모던하고 클린한 느낌

3. **강한 대비**
   - 검정 (#000) vs 흰색 (#fff)
   - 명확한 시각적 계층
   - 가독성 최우선

4. **미니멀리즘**
   - 불필요한 장식 제거
   - 넓은 여백
   - 단순한 레이아웃
   - 깔끔한 타이포그래피

5. **그리드 시스템**
   - 정렬된 레이아웃
   - 일관된 간격
   - 정돈된 구조

6. **미묘한 그림자**
   - 과하지 않은 그림자
   - 깊이감 최소화
   - Flat 디자인 지향

7. **상호작용**
   - Hover: 약간 어두운 회색
   - Active: 스케일 축소 (0.98)
   - Focus: 검정 테두리
   - Disabled: 밝은 회색

8. **타이포그래피**
   - Sans-serif 시스템 폰트
   - Letter spacing 조정
   - Weight 대비로 계층 구분
   - 검정-회색 대비

=========================================
[접근성]
=========================================

1. **색맹 대응**
   - 무채색이므로 완벽 대응
   - 색상만으로 정보 전달 안 함
   - 텍스트, 아이콘 병행

2. **명도 대비**
   - WCAG AAA 준수
   - 검정 vs 흰색 = 21:1
   - 회색 조합도 4.5:1 이상

3. **키보드 네비게이션**
   - Focus 상태 명확 (검정 테두리)
   - Tab 순서 논리적
   - Skip to content

4. **스크린 리더**
   - Semantic HTML
   - ARIA labels
   - Alt text

[개발 가이드라인]
=========================================

1. **색상 사용 금지 목록**
   - 절대 사용 금지: 빨강, 파랑, 녹색, 노랑 등 모든 유채색

   - 모든 border-radius: 0
   - 예외: spinner (50%)

3. **그림자**
   - Box-shadow: rgba(0, 0, 0, 0.08 ~ 0.3)
   - 과도한 그림자 금지

4. **버튼**
   - Primary: #000 배경, 흰 텍스트
   - Secondary: 흰 배경, #000 테두리
   - Disabled: #ccc 배경

5. **입력 필드**
   - 배경: #fafafa
   - 테두리: #e5e5e5
   - Focus: #000 테두리

6. **타이포그래피**
   - 제목: #000 ~ #1a1a1a
   - 본문: #333 ~ #666
   - 보조: #888 ~ #aaa
