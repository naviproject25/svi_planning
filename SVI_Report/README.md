# SVI 설문 시스템 - 무채색 모던 디자인

사회적가치지표 측정 설문 및 보고서 시스템

## 빠른 시작

### 파일 구조
```
├── login.html          # 로그인 (완료)
├── survey.html         # 설문 입력 (3/15 페이지 완료)
├── admin-list.html     # 관리자 리스트 (완료)
├── report.html         # 보고서 (완료)
├── 기획서.txt          # Figma Design용 기획서
├── SESSION_CONTEXT.md  # 세션 컨텍스트 (자세한 정보)
└── Src/                # 참조 이미지
```

## 디자인 컨셉
- **완전 무채색** (흑백 + 그레이스케일)
- **직각 디자인** (border-radius: 0)
- **미니멀리즘**
